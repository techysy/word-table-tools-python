# 📝 Word 表格批量工具

批量替换和格式化 Word 文档中的表格内容，让繁琐的文档处理变得轻松高效！

## ✨ 功能

- 🔄 **内容替换** - 批量替换表格中的指定文字（支持精确匹配和包含匹配）
- 🎨 **格式刷** - 从模板文件复制表格格式到其他文件
- 📋 **日志记录** - 详细记录每次操作，方便排查问题

## 🚀 使用方法

### 1. 🔄 内容替换工具

双击 `启动替换工具.bat` 运行

1. 在"查找"和"替换"输入框填写规则
2. 拖拽 Word 文件到窗口，或点击添加文件
3. 点击"执行替换"

### 2. 🎨 格式刷工具

双击 `启动格式刷.bat` 运行

1. 选择一个格式正确的模板文件
2. 添加需要修改格式的目标文件
3. 点击"执行格式复制"

## 🎯 匹配规则

- 🔍 **包含匹配** - 单元格内容包含查找文字即替换
- 🛡️ **自动排除表头** - 长文本（>50字符）自动跳过，避免误替换

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `word_replacer_tool.py` | 🔄 内容替换工具 |
| `format_brush.py` | 🎨 格式刷工具 |
| `logs/` | 📋 操作日志 |
